THE TRUTH

Never write checks for true / false again!

usage:

```
npm i -S the-truth

var theTruth = require('the-truth')

var iAmTryingToBreakTheInternet = true

theTruth(iAmTryingToBreakTheInternet) // it's TRUE!

```
